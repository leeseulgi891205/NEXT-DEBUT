PROJECT SUMMARY

아이돌 육성 세계관과 커뮤니티 경험을 결합한 웹 프로젝트

NEXT DEBUT는 연습생 성장, 상호작용형 게임 흐름, 커뮤니티 요소, 결제 및 인증 흐름까지 하나의 서비스 경험으로 엮은 포트폴리오 프로젝트입니다.

STACK

  • Spring Boot 4 / JSP / Gradle
  • Spring Security / OAuth2 / WebSocket
  • H2 Database / JPA
  • FastAPI 기반 ML 보조 예측

CORE HIGHLIGHTS

포트폴리오에서 강조할 만한 포인트

게임형 사용자 경험

정적인 소개 페이지가 아니라 사용자가 서사와 선택지를 따라가며 몰입할 수 있는 구조를 담았습니다.

커뮤니티와 팬 경험

가이드, 게시판, 팬미팅 흐름 등 실제 서비스 감각을 포트폴리오 안에 녹였습니다.

AI/ML 연동 설계

Python ML 보조 예측과 채팅 로직이 함께 동작하도록 구성해 기술적 확장성을 보여줍니다.
